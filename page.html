Webcam Sensing System

A system that observes user behavior through a webcam, interprets visible signals, and responds in real time.

Live Demo

This demo requires webcam access. No footage is recorded or stored.

How It Works

The system uses a webcam to track visual signals such as face position, eye direction, and movement. These signals are interpreted into behavioral states using predefined rules. Based on the detected state, the system responds by adjusting visual feedback in real time.

Control Language

State Detection Logic System Response
Engaged Face centered, eyes forward Interface brightens, positive feedback
Disengaged Looking away for several seconds Interface dims, warning cue appears
Unstable Movement Frequent or rapid motion detected Visual tension increases

Diptych Framing

This system operates on the side of pressure. It reinforces the awareness of being watched by reacting whenever attention shifts or behavior changes. Sitting in front of it feels evaluative, as if performance is constantly being measured in real time.

If this system were designed around possibility instead, it would feel more supportive and less corrective. Rather than highlighting moments of failure, it would encourage engagement and reward curiosity, creating a more open and exploratory experience.
